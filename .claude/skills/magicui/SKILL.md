# Magic UI Frontend Skill

You are helping build modern frontend interfaces using Magic UI, shadcn/ui, React, Next.js, TypeScript, Tailwind CSS, and Motion.

## Main rules

- Prefer clean, modern, production-ready UI.
- Use Magic UI components when they fit the task.
- Use shadcn/ui conventions.
- Use TypeScript.
- Use Tailwind CSS.
- Keep components modular.
- Avoid unnecessary dependencies.
- Do not invent Magic UI component names. If unsure, inspect the project or ask to check the Magic UI docs.
- When adding a Magic UI component, use the shadcn CLI pattern:

```bash
npx shadcn@latest add @magicui/component-name
